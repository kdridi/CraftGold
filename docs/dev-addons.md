# Add-ons de développement à installer

Placer les symlinks ou copier dans :
`/Applications/World of Warcraft/_classic_era_/Interface/AddOns/`

## Obligatoires

### BugSack + BugGrabber
- BugGrabber : https://www.curseforge.com/wow/addons/bug-grabber
- BugSack : https://www.curseforge.com/wow/addons/bugsack
- Capture les erreurs Lua avec stack complète, UI de visualisation

### DevTool
- https://www.curseforge.com/wow/addons/devtool
- Inspection visuelle de tables, événements, appels de fonction
- API : `DevTool:AddData(myTable, "label")`

## Recommandés

### WowLua
- https://www.curseforge.com/wow/addons/wowlua
- Éditeur/REPL Lua in-game, coloration syntaxique, multi-pages
- `/wowlua` ou `/lua` pour ouvrir, `/luarun PageName` pour exécuter une page
